# Website for shorten long URL
## Demo: <a href="https://shorten.rf.gd">shorten.rf.gd</a>
## How to shorten URL:
- Enter URL
- Enter Alias (Short Code)
- Click Shorten <br>
  Example: <br>
    - URL: Google.com <br>
    - Alias: google
## How to visit shortened URL:
- Enter (Domain name)/(Alias) in search bar <br>
  Ex: <a href="https://shorten.rf.gd/google">shorten.rf.gd/google</a> <br>
  will be redirected to => google.com
